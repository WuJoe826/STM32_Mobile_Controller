## 簡介：
！！！項目仍然在初始階段，請見諒！！！
> 使用STM32微控制器的Type-C掌上手機游戲手柄，使用微軟X-input API，支持大部分Type-C手機。
## 功能：
- 綫性扳機鍵
- 2個功能按鍵
- 十字鍵
- ABXY鍵
- 體感陀螺儀（模擬鼠標）
- 肩鍵
- 搖桿
- 2個自定義功能鍵
